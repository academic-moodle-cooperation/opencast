Opencast 5: Release Notes
=========================

New Features and Improvements
-----------------------------

- Paella Player
- Java 8
- Composer updates
- OAI-PMH
- Duplicate, clone operation
- Moodle user/role provider
- Performance improvements


Configuration changes
---------------------

- HTTP Basic authentication is enabled by default
